Colour Palette:

Background: bg-background
Primary Text: text-text
Secondary Text: text-link (we'll use this for less emphasised text)
Accent 1: text-accent1 (for highlights and important elements)
Accent 2: text-accent2 (for buttons/CTAs)
Highlight: text-highlight (for hover effects and emphasis)

Font Family:

Primary (body text): Roboto
Secondary (headings): Montserrat
Accent (code snippets, tech stack): Chivo Mono

Style Guide:

Home Page:

Background: bg-background
Name: font-secondary, text-4xl or text-5xl, text-text, font-bold
Job Title: font-secondary, text-2xl, text-link
Description: font-primary, text-lg, text-text
Navigation: Position on right, font-primary, text-sm, text-link, hover:text-accent1
Social Icons: text-link, hover:text-accent2

Projects Section:

Section Title: font-secondary, text-3xl, text-text, mb-6
Project Cards:

Card Background: bg-background (slightly lighter shade if possible)
Image/GIF: 70% of card height
Project Name: font-secondary, text-xl, text-text, font-semibold
Tech Stack: font-accent, text-sm, text-link
Language Icons: text-accent2

Project Modal:

Background: bg-background with opacity
Modal Content: bg-background (slightly lighter shade), rounded-lg, p-6
Title: font-secondary, text-2xl, text-text, mb-4
Description: font-primary, text-text
Tech Details: font-accent, text-sm, text-link
Close Button: absolute top-right, text-link, hover:text-accent1

Detailed Project Page:

Table of Contents:

Position: Left side, sticky
Font: font-primary, text-sm, text-link
Active Link: text-accent1

Content:

Headers: font-secondary, text-2xl/text-xl/text-lg, text-text
Body Text: font-primary, text-text
Code Snippets: font-accent, bg-background (darker shade), p-4, rounded

Images: max-width-100%, rounded-lg, my-4

For the detailed project page content, consider including:

Project Overview: Brief description and goals
Technical Challenges: Interesting problems you solved
Design Decisions: Why you chose certain technologies or approaches
Key Features: Highlight the most impressive or innovative aspects
Development Process: Brief insight into your workflow or methodology
Lessons Learnt: What you gained from the project
Future Improvements: Ideas for expanding the project
Demo/Live Link: If applicable
GitHub Repository: Link to the source code
